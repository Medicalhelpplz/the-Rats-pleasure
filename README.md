## Installation

Requirements are Yarn and Node.js (v16+).

```
git clone https://github.com/titaniumnetwork-dev/Oxide-Docs.git
cd Oxide-Docs
yarn

yarn start # developmental server
yarn build # build
yarn serve # production build
```
